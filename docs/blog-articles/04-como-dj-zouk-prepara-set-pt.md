## TEXTO DO ARTIGO (colar no WordPress):

### Como um DJ de Zouk prepara um set de festival de 3 horas

Existe uma pergunta que os dançarinos raramente fazem, mas que define completamente a qualidade do que eles vão sentir na pista: o que acontece nos dias antes de um festival começar?

Vou te contar o que acontece do meu lado.

**O que um set de 3 horas realmente exige**

Um set de 3 horas de Zouk não é uma playlist de 3 horas. É uma experiência com começo, desenvolvimento e conclusão — uma jornada emocional com arco dramático, como um livro ou um filme.

Para que isso funcione, cada decisão precisa ser tomada antes de a primeira música começar. Improvisar completamente é possível para DJs com décadas de coleção na memória — mas mesmo esses DJs estão tomando decisões que foram treinadas ao longo de anos.

**Fase 1 — Pesquisa e curadoria (semanas antes)**

A preparação de um set começa muito antes do festival. Nos dias que antecedem, estou:

- **Ouvindo produções novas** em Spotify, SoundCloud e plataformas de produção de Zouk — qualquer coisa lançada nos últimos 3–6 meses que possa ter passado despercebida
- **Revisitando clássicos** que funcionam em certos contextos (amanheceres, picos de pista, momentos de construção emocional)
- **Analisando o público do festival** — qual é o nível técnico predominante dos dançarinos? Qual é a temperatura emocional que esse festival costuma ter? Isso muda profundamente a seleção

**Fase 2 — Estrutura do set (dias antes)**

Com as músicas em mente, construo a arquitetura do set. Não é uma lista de ordem — é uma estrutura emocional.

Um set típico de festival tem aproximadamente:

| Segmento | Duração | Objetivo emocional |
|---|---|---|
| Abertura | 20–30 min | Acolhimento, convite ao fluxo |
| Construção | 40–50 min | Aprofundamento, conexão |
| Pico | 30–40 min | Energia máxima, catarse |
| Descida | 20–30 min | Resolução emocional |
| Fechamento | 10–20 min | Memória, despedida |

Cada música que escolho para cada segmento precisa fazer sentido dentro dessa curva. Uma música que seria perfeita para o pico pode destruir a abertura.

**Fase 3 — Preparação técnica (dia antes)**

Com a estrutura definida, faço a preparação técnica:

- **BPM mapping** — garanto que as músicas de cada segmento têm BPMs que permitem transições suaves
- **Key matching** — verifico a compatibilidade harmônica entre músicas que vão estar próximas no set (transições entre músicas na mesma tonalidade ou em tonalidades relacionadas soam mais coesas)
- **Teste de transições críticas** — as 5–10 transições mais importantes do set são testadas antes, especialmente as que ocorrem nos momentos de maior mudança emocional

**Fase 4 — Adaptação em tempo real**

Tudo que preparei é um mapa — não um roteiro fixo. A habilidade real do DJ é ler o que está acontecendo na pista e adaptar:

- O casal que para de dançar quando determinada música começa — sinal de que o segmento terminou antes do planejado
- O grupo que claramente quer mais energia quando eu estava planejando desacelerar
- O momento em que toda a pista sincroniza e o silêncio depois de uma transição perfeita é preenchido pelo som dos dançarinos sorrindo

Esses sinais só se aprendem tocando. E tocando muito.

**Por que isso importa para os dançarinos**

Quando um set funciona, ninguém nota o trabalho que estava por trás. As músicas parecem ter se escolhido sozinhas, as transições parecem inevitáveis, a noite parece ter tido exatamente a forma que deveria ter.

Quando um set não funciona, todos sentem — mas raramente conseguem nomear o que deu errado. A sensação é de que "a noite não emplacou", "as músicas eram estranhas", "não entrei no ritmo".

A diferença entre essas duas experiências é exatamente o que acontece nos dias antes de a festa começar.
