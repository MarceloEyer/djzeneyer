# SETUP.md - Guia de Instalacao

## Pre-requisitos

- Node.js 20+
- PHP 8.3+
- WordPress 6.9+
- MariaDB ou MySQL compativel com WordPress e WooCommerce

## Plugins essenciais

- WooCommerce
- GamiPress
- Polylang
- LiteSpeed Cache

## Plugins customizados deste repo

- `plugins/zeneyer-auth`
- `plugins/zen-seo-lite`
- `plugins/zen-bit`
- `plugins/zengame`

## Fluxo basico

```bash
npm install
npm run lint
npm run build
```

## Nota

Este guia e operacional. Regras canonicas ficam em `AI_CONTEXT_INDEX.md` e no contexto local do projeto.
