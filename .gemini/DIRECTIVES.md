# Zen Eyer AI Directives
Version: 1.0.0

Este arquivo contém instruções específicas para agentes de IA que operam neste repositório.

## Regras de Ouro
1. **Identidade:** O nome oficial é `Zen Eyer`. O alias é `DJ Zen Eyer`. O termo `Zen Ayer` está proibido em metadados.
2. **SSOT:** Consulte sempre `src/data/artistData.ts` antes de sugerir ou alterar textos sobre o artista.
3. **Fluxo:** Nunca use `git push` na `main`. Crie branches e PRs para cada tarefa.
4. **Build:** Rode `npm run lint` antes de propor qualquer mudança em componentes React.
5. **Comunicação:** Pergunte ao usuário antes de tomar decisões de design ou copy.

Consulte `DOCS_ENGINEERING.md` na raiz para o guia completo de padrões.
