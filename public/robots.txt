# ==============================================================================
# 📘 DOCUMENTAÇÃO: O QUE É CADA DIRETIVA
# ==============================================================================
# 
# User-agent: Define qual bot/crawler esta regra se aplica
#   - Use "*" para todos os bots
#   - Use nome específico (ex: "Googlebot") para regras individuais
#   - Múltiplos User-agent seguidos aplicam as mesmas regras
#
# Allow: Permite acesso explícito a um caminho
#   - Útil para liberar subdiretórios dentro de um Disallow
#   - Exemplo: Allow: /wp-admin/admin-ajax.php
#
# Disallow: Bloqueia acesso a um caminho
#   - Disallow: / = bloqueia tudo
#   - Disallow: (vazio) = permite tudo
#   - Usa wildcards: * (qualquer string) e $ (fim da URL)
#
# Crawl-delay: Tempo mínimo (em segundos) entre requisições
#   - NÃO suportado por Google/Bing (use Google Search Console)
#   - Suportado por Bing, Yandex e alguns bots menores
#   - Valor recomendado: 1-10 segundos
#
# Sitemap: URL do sitemap XML
#   - Obrigatório para SEO
#   - Pode ter múltiplas linhas Sitemap
#
# ==============================================================================

Disallow: /*?*sort=*          # Parâmetros de ordenação
Disallow: /*?*filter=*        # Parâmetros de filtro
