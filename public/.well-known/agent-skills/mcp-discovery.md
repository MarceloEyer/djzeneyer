# MCP Discovery

Use this skill when an agent or MCP client needs to discover Zen Eyer public context resources through MCP.

Server card:

`https://djzeneyer.com/.well-known/mcp/server-card.json`

MCP endpoint:

`https://djzeneyer.com/wp-json/djzeneyer/v1/mcp`

Authentication: none required for public resources.
