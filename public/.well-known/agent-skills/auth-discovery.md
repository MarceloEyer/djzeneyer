# Auth Discovery

Use this skill when an agent needs to understand authentication for Zen Eyer public resources.

Public AI discovery resources do not require authentication. Optional public-read registration is documented at:

`https://djzeneyer.com/.well-known/auth.md`

OAuth-style metadata:

- `https://djzeneyer.com/.well-known/oauth-protected-resource`
- `https://djzeneyer.com/.well-known/oauth-authorization-server`
